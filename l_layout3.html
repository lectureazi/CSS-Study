position : 요소가 배치되는 위치를 지정하는 속성

            position:static   : 기본 값.  브라우저가 배치하는 위치에 요소가 배치된다.
                                요소의 위치를 지정하기 위한, top, bottom, left, right, z-index 속성을 사용할 수 없다.
            
            position:relative : 브라우저가 배치하는 위치에 요소가 배치된다.
                                요소의 위치를 지정하기 위한, top, bottom, left, right, z-index 속성을 사용할 수 있다.

            position:absolute : 요소의 위치를 지정하기 위한, top, bottom, left, right, z-index 속성을 사용할 수 있다.
                                position:absolute 인 요소는 normal flow에서 제외된다.
                                absolute를 top,bottom,left,right를 사용해 이동시킬 경우 부모 요소의 상하좌우 위치가 기준이 된다.
                                ex) relative : left 5%  => 자신의 위치로부터 우측으로 부모요소의 5%만큼 이동
                                    absolute : left 5%  => 부모요소의 좌측 경계를 기준으로 우측으로 부모요소의 5%만큼 이동

            position:fixed    : 스크롤과 상관없이 화면의 좌측상단을 기준으로 요소가 배치된다.
                                요소의 위치를 지정하기 위한, top, bottom, left, right, z-index 속성을 사용할 수 있다.

top, bottom, left, right

            top : 위에서 아래로
            bottom : 아래에서 위로
            left : 왼쪽에서 오른쪽으로
            right : 오른쪽에서 왼쪽으로

            position과 top, bottom, left, right 속성을 사용해 요소를 %로 이동시키면 부모요소의 높이와 너비를 기준으로 이동하게 된다.
rel, right:5%
rel, top:5%
rel, right:5%; top:5%
rel left:6%; bottom:2%
abs, right:5%
abs, top:5%
abs, right:5%; top:5%
abs left:6%; bottom:2%
